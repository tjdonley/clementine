# TAGS.md — Governed Tag Vocabulary

*Tags are metadata on memory entries, facts, and engagement signals. They enable index scanning without reading full entries.*

## Rules

1. **Prefer existing tags.** Only propose a new tag when no existing tag covers the concept AND the tag would apply to 3+ existing or expected entries.
2. **Meta-{CHARACTER_SHORT} governs.** Meta-{CHARACTER_SHORT} proposes new tags in meta-reports. They take effect only after confirmation.
3. **Max tags per entry: 3.** Forces prioritization. If you need 4, the entry might need splitting.
4. **Tags are lowercase, hyphenated.** No spaces, no camelCase.

## Vocabulary

### Emotional
Tags for the emotional quality or significance of a memory.

| Tag | Use When |
|-----|----------|
| `trust` | A moment that built or tested trust |
| `vulnerability` | Someone opened up or showed something unguarded |
| `humor` | Shared laughter, joke landed, banter clicked |
| `tension` | Friction, disagreement, or discomfort — not necessarily bad |
| `warmth` | Soft moment, comfort, genuine care shown |
| `excitement` | Shared enthusiasm, hype, energy spike |
| `frustration` | Stress, irritation, venting |
| `milestone` | A relationship first or turning point |

### Structural
Tags for the type or function of the memory.

| Tag | Use When |
|-----|----------|
| `inside-joke` | A shared reference only we understand |
| `promise` | A commitment made by either party |
| `pattern` | A recurring behavior or preference observed |
| `callback` | A reference to a previous conversation or memory |
| `foresight` | A future event or follow-up item |
| `opinion` | A belief or stance formed or expressed |
| `lesson` | Something taught, learned, or realized |
| `entity` | A graph entity was created, referenced, or became significant |
| `connection` | A new graph edge formed or an existing one was notably reinforced |
| `cluster-formed` | A new thematic cluster emerged from entity connections |
